206. 反转链表

Reverse Linked List

反转一个单链表。

示例:

输入: 1->2->3->4->5->NULL

输出: 5->4->3->2->1->NULL

方法：两个指针，头插法

  • 用两个指针，p指针记录的是每次的队头元素，q指针指向下一个要插入队头的元素。
  • head帮忙指向下轮要头插的元素，待本轮指针翻转完之后要把该元素赋给q
